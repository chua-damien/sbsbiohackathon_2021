Documents in this folder: 

1. _code for DEG analysis: contains code for DEG analysis
2. _code for remapping reads: pynb for remapping using InterProScan
3. all_data.RData : R environment to be loaded for Shiny R app.
4. app.R : App script to run the Shiny R app. 
5. homepage.html : homepage in html format for Shiny R app.
6. server.R : server script for R shiny app. To be run prior to running the app.R
7. ui.R: user interface script for R shiny app. To be run prior to running the app.R
